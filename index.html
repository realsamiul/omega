üé® CSS Architecture

CSS is loaded in the following cascade order:

  1. fonts.css - Font-face definitions
  2. main.css - Base styles & resets
  3. layout.css - Grid system
  4. components.css - Component styles
  5. animations.css - Animations
  6. Locomotive Scroll CSS (CDN)

‚ö° JavaScript Loading

Scripts load in this order for proper initialization:

  1. GSAP (CDN)
  2. ScrollTrigger (CDN)
  3. Locomotive Scroll (CDN)
  4. boot-animations.js - Initializes scroll system
  5. app.js - Main application
